## 1. csv

- 콤마로 데이터 구분
- csv파일이 깨질 때 text로 열어서 인코딩 UTF-8로 저장

```jsx
출판연도, 제목, 매출(만원), 순익(만원), 가격(원);
2020, Python, 300, 10, 1000;
2021, HTML, 310, 20, 2000;
2022, CSS, 320, 30, 3000;
2023, JavaScript, 330, 40, 4000;
2024, D3, 340, 10, 5000;
```

## 2. tsv

- 탭으로 데이터 구분

```jsx
출판연도	제목	매출(만원)	순익(만원)	가격(원)
2020	Python	300	10	1000
2021	HTML	310	20	2000
2022	CSS	320	30	3000
2023	JavaScript	330	40	4000
2024	D3	340	10	5000
```
